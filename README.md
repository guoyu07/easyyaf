## 框架分类
1）针对中小型公司推荐框架 --- easyYaf!<br>
2）针对大中型公司(web与service分离，分布mysql，分布redis，分布memcache、swoole、workerman等)推荐框架 -- hardYaf!<br>
3）针对移动端接口（支持rewrite/cli/restful）推荐框架 --apiYaf<br>
